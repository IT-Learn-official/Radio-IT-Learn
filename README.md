# Radio-IT-Learn

A website showcasing curated playlists from multiple streaming platforms, designed for developers and tech enthusiasts.

## Features

- **Multiple Streaming Platforms**: Browse playlists from Spotify, YouTube, Apple Music, SoundCloud, and Deezer
- **Filter by Platform**: Easily filter playlists by your preferred streaming service
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Developer-Focused Content**: Curated playlists specifically designed for coding, learning, and productivity

## Getting Started

Simply open `index.html` in your web browser to get started. Click "Explore Playlists" to view all available playlists.

## File Structure

- `index.html` - Landing page with hero section
- `playlists.html` - Main playlists page with filtering functionality

## Usage

1. Open `index.html` in your browser
2. Click on "Explore Playlists" to view all available playlists
3. Use the platform filters to narrow down playlists by streaming service
4. Click "Listen Now" on any playlist to open it in the respective streaming platform
